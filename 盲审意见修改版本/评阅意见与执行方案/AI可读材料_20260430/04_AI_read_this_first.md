# AI read-this-first file

Recommended reading order for future AI assistants:

1. `00_README.md` for provenance and package structure.
2. `01_thesis_before_pages.jsonl` for page-by-page access to the pre-revision thesis.
3. `02_review_return_pdf_pages.jsonl` for page-by-page access to the official returned review PDF.
4. `03_review_comments_txt_utf8.md` for the UTF-8 version of the review/comment plan text.
5. `03_review_comments_txt_sections.jsonl` when section-level chunking is needed.

Notes:

- `*_full_layout.txt` keeps PDF line layout as much as possible.
- `*_full_plain.txt` is easier for continuous reading.
- `*.jsonl` files are best for scripts, RAG, and page/section-based reading.

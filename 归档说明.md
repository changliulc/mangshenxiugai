# 论文版本归档说明

本文件夹用于集中保存论文旧版本与盲审意见修改版本。归档时仅执行复制操作，原有目录未移动、未删除。

## 目录结构

### 旧版本

```text
旧版本/
  明审版本/
  原始PDF/
  第三章原始代码与结果_20260304/
```

来源如下：

```text
明审版本/
刘畅_23011211000_面向智慧交通的车型分类及停车占用检测方法研究_1.pdf
非核心文件/绘图/第三章/20260304/
```

其中，第三章原始论文结果对应的最终输出包位于：

```text
旧版本/第三章原始代码与结果_20260304/out_ch3_revision_pkg_20260314_seed42_v8/
```

生成该结果包的主要脚本为：

```text
旧版本/第三章原始代码与结果_20260304/build_ch3_revision_pkg.py
```

### 盲审意见修改版本

```text
盲审意见修改版本/
  明审版本/
  评阅意见与执行方案/
```

来源如下：

```text
论文返回意见修改/明审版本/
修改意见.txt
完整修改意见与执行方案.md
AI_readable_thesis_before_and_review_20260430/
```

当前第三章补充实验代码与正式补充实验结果位于：

```text
盲审意见修改版本/明审版本/第三章第一类补充实验代码/
```

正式建议使用的补充实验结果目录为：

```text
盲审意见修改版本/明审版本/第三章第一类补充实验代码/outputs/20260503_第5次实验/
```

## 使用提醒

- `旧版本` 用于保留原论文、原始代码和原始结果。
- `盲审意见修改版本` 用于后续继续修改正文、补充实验结果和整理修改说明。
- 不建议直接在本归档中继续频繁修改实验代码；若继续迭代，可先复制出新的工作副本。
